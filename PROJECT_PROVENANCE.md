## Project Purpose

LedgerShield_AI™ is a financial-risk intelligence and banking-security dashboard built for portfolio, academic, and professional demonstration.

The project includes a Next.js frontend, FastAPI backend, synthetic financial-risk data generation, customer risk profiling, model metrics, DDoS simulation, traffic streaming, and deployment-ready configuration.
